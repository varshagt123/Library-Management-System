# 📚 Library Management System

## Overview
This is a web-based Library Management System developed using PHP, MySQL, HTML, CSS, JavaScript, PHPMailer, and FPDF. It helps manage books, members, transactions, fines, and email reminders.

## Features
- User Login
- Book Management
- Member Management
- Issue and Return Books
- Fine Calculation
- Email Reminder using PHPMailer
- PDF Report Generation using FPDF

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- PHPMailer
- FPDF
- XAMPP

## Database
Import the `library_db.sql` file into phpMyAdmin before running the project.

## Installation
1. Install XAMPP.
2. Copy the `LibraryMs` folder into the `htdocs` directory.
3. Start Apache and MySQL.
4. Import `library_db.sql` into phpMyAdmin.
5. Open:
   ```
   http://localhost/LibraryMs
   ```
